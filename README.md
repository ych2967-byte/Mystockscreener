# 무료 일봉 주식 스크리너

아이폰에서 사용하는 한국·미국 주식 스크리너입니다.

- 비용: 0원
- API 키: 필요 없음
- 입력: CSV 불필요
- 화면: GitHub Pages 정적 웹사이트
- 자동 갱신: GitHub Actions
- 가격 자료: yfinance가 읽는 공개 Yahoo Finance 데이터
- 한국 종목 목록: 네이버 금융 시장 목록
- 미국 1차 범위: S&P 500 + Nasdaq-100

아이폰 설치 방법은 [`START_HERE_IPHONE.md`](START_HERE_IPHONE.md)를 확인하세요.

## 데이터 구조

자동 실행 프로그램이 종목별 약 18개월의 일봉을 읽고 필요한 숫자만 계산하여 아래 파일에 저장합니다.

- `docs/data/kr.json`
- `docs/data/us.json`

웹사이트는 이 두 파일만 읽기 때문에 아이폰에서도 가볍게 작동합니다.

## 2차 확장: 미국 일반주 전체

2차 버전에서는 Nasdaq Trader의 상장 종목 목록을 이용하여 Nasdaq·NYSE·NYSE American 일반주를 모으고, ETF·워런트·권리·테스트 종목을 제거할 수 있습니다. 가격 요청은 100~200종목 단위 여러 작업으로 나눠야 무료 데이터 제공처의 요청 제한 위험을 줄일 수 있습니다.

## 면책

무료 공개 데이터는 지연, 누락, 수정 가능성이 있습니다. 이 프로젝트는 개인 연구용이며 투자 권유나 주문 시스템이 아닙니다.
