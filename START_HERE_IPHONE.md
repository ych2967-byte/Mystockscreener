# 아이폰에서 시작하기

이 프로젝트는 **API 키 없이**, **CSV 업로드 없이**, **하루 한 번 자동 갱신**되는 무료 일봉 스크리너입니다.

## 완성 후 사용법

사파리에서 사이트 주소를 열고 `공유 → 홈 화면에 추가`를 누르면 앱처럼 사용할 수 있습니다.

## 처음 설치할 때 필요한 것

- 아이폰
- 무료 GitHub 계정
- 이 압축파일
- 공개 저장소 1개

GitHub Pages 무료 기능은 공개 저장소에서 사용합니다. 투자 계좌나 증권사 API 키는 연결하지 않습니다.

## 설치 순서

### 1. GitHub에 새 저장소 만들기

1. 사파리에서 GitHub에 로그인합니다.
2. 오른쪽 위 `+`를 누르고 `New repository`를 선택합니다.
3. 저장소 이름을 `my-stock-screener`로 입력합니다.
4. `Public`을 선택합니다.
5. `Add a README file`은 켜지 않습니다.
6. `Create repository`를 누릅니다.

### 2. 압축을 풀고 파일 올리기

1. 아이폰 `파일` 앱에서 ZIP을 누르면 같은 이름의 폴더가 생깁니다.
2. GitHub 저장소에서 `Add file → Upload files`를 누릅니다.
3. 압축을 푼 폴더 안의 파일과 폴더를 올립니다.

아이폰에서 폴더 전체 선택이 어렵다면 ChatGPT에 현재 화면을 캡처해 보내주세요. 그 화면을 기준으로 다음 버튼을 안내받을 수 있습니다.

### 3. 사이트 켜기

1. 저장소 위쪽 `Settings`를 누릅니다.
2. 왼쪽 메뉴에서 `Pages`를 누릅니다.
3. `Build and deployment`의 `Source`를 `Deploy from a branch`로 둡니다.
4. Branch는 `main`, 폴더는 `/docs`를 선택합니다.
5. `Save`를 누릅니다.

잠시 뒤 같은 화면 위쪽에 사이트 주소가 표시됩니다.

### 4. 첫 데이터 갱신 실행

1. 저장소 위쪽 `Actions`를 누릅니다.
2. `한국주식 데이터 갱신`을 선택합니다.
3. `Run workflow → Run workflow`를 누릅니다.
4. 같은 방법으로 `미국주식 데이터 갱신`도 실행합니다.

이 첫 실행 뒤부터 평일마다 자동으로 갱신됩니다.

## 갱신 시간

- 한국주식: 한국시간 평일 오후 5시 30분
- 미국주식: 미국 장 마감 뒤, 한국시간 화~토 오전 6시 30분 전후

GitHub의 무료 예약 실행은 사람이 정한 시각보다 몇 분 늦게 시작될 수 있습니다.

## 들어 있는 조건

- 5일선 > 20일선
- 20일선 > 50일선
- 50일선 > 100일선
- 100일선 > 200일선
- **5일선 > 20일선 > 50일선 > 100일선 > 200일선 전체 정배열**
- 현재가 > 200일선
- 오늘·1주·1개월·3개월 상승률
- 20일 평균 대비 거래량 증가율
- 20일선 이격률
- 20일 고점까지 남은 거리

## 알아둘 점

이 프로그램은 무료 공개 데이터를 개인 연구 목적으로 사용합니다. 일부 종목 누락이나 지연이 생길 수 있으므로 실제 주문 전 가격은 증권사 앱에서 다시 확인해야 합니다.
